Project 2

EVM using FPGA Numato Mimas Version 2

Description:Polling by Electronic Voting Machine (EVM) is easy, safe and secure methodology that takes minimum of our time .In order to perform this mechanism, there were several phases in the design process such as designing a flow chart, algorithm and simultaneously the code is developed to implement & stimulate the logic. The proposed digital EVM was designed on Xilinx ISE using verilog HDL and can also be implemented on FPGA board for real time purpose.

Software used: Xilinx Programing Language,FPGA kit

Project 3

Designing of Horn Antenna Using HFSS

Description:The horn was designed by implementing a procedure in MathCAD. Simulation & optimization of the horn was carried out in HFSS based on finite element method (FEM). The reason to simulate the horn in HFSS is the fact to achieve optimum gain and equal radiation patterns in both E-plane & H-plane (pencil beam).

Software used: HFSS Software
